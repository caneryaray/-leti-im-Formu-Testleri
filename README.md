Proje Hakkında – Örnek Açıklama

Bu proje, React kullanılarak geliştirilmiş bir İletişim Formu Uygulamasıdır. Kullanıcılar form aracılığıyla ad, soyad, e-posta ve mesaj bilgilerini girip gönderebilir. Form, girilen bilgileri anında kontrol ederek hataları kullanıcıya bildirir ve yalnızca geçerli bilgiler gönderildiğinde ekranda görüntüler.

Projenin amacı, React ile form yönetimi, validation ve kullanıcı deneyimi odaklı bir uygulama geliştirmektir. Bu sayede form işlemleri sırasında hataların nasıl yönetileceği, state kullanımı ve componentler arası veri aktarımı öğrenilmiş olur.

Uygulama, Goruntule adında ayrı bir component ile form verilerini ekranda gösterir. Ayrıca, Vitest ve React Testing Library kullanılarak yazılmış testlerle formun tüm işlevleri otomatik olarak kontrol edilir. Bu testler, formun doğru şekilde render edilmesini, hataların düzgün gösterilmesini ve gönderilen verilerin doğru şekilde görüntülenmesini garanti eder.

Bu proje sayesinde:

React ile component tabanlı form geliştirme

State yönetimi ve validation uygulama

Test yazımı ve test odaklı geliştirme (TDD)
gibi temel frontend beceriler pekiştirilebilir.
